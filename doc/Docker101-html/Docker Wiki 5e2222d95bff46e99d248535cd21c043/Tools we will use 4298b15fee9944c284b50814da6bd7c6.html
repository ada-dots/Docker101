Tools we will use

Operating system that we cau use is either Windows 10 or Linux or Mac

Tools we will use:

  • Windows Subsystem for Linux
    • On Windows 10, enable WSL, by following the steps described here: Enable WSL
    • Add the Ubuntu distro from the Microsoft Store
  • Visual Studio 2019 / VS Code
    • First, you will need to have a development IDE installed. The most preferable IDE is Visual Studio 2019  if you are running the Windows operating system.
    • You can also use Visual Studio Code .
      • A couple of VS Code extensions are required to assist you during the labs. You can install them before the workshop or just in time.
      • On Windows, install the Remote WSL extension : lets you use VS Code on Windows to build Linux applications that run on the Windows Subsystem for Linux (WSL).
      • Install the Docker extensions
        • and configure VS Code for debugging C# by installing the C# extension
  • Docker Desktop (Windows & Mac)
    • you are going to need the Docker Community Edition tooling on your development machine.
    • Download and install Docker Community Edition (version 3.1.0 or higher):
      • Docker Desktop for Windows
      • Docker Desktop for Mac
    • When on Windows, make sure to enable WSL integration, if possible.
    • .NET Platform
      • Download and install .NET 5  if needed.
    • Windows Terminal
      • The Windows Terminal is a modern, fast, efficient, powerful, and productive terminal application for users of command-line tools and shells like Command Prompt, PowerShell, and WSL
